## 🧠 What is this?

**Vibe coding** is the art of building real, production-ready software using AI tools — without needing a traditional CS degree. Tools like Claude, Cursor, v0, Copilot, and ChatGPT have made this possible for thousands of developers worldwide.

But there's a gap: **most people don't know how to prompt effectively.**

This repo closes that gap. It's a structured, community-maintained library of prompts and templates that help you:

- 🚀 Scaffold full projects from scratch
- 🐛 Debug faster and smarter
- 🎨 Generate clean UI with AI tools
- 🔍 Review and refactor your code
- 🚢 Deploy confidently

## 🚀 Quick Start

1. Browse the `prompts/` folder for your use case
2. Copy the prompt
3. Paste it into Claude, Cursor, ChatGPT, or your AI tool of choice
4. Replace `[BRACKETED]` placeholders with your specifics
5. Ship it ✅

## ✨ Featured Prompts

### 🏗️ Full Stack Project Scaffolder
```
I want to build [PROJECT DESCRIPTION].

Tech stack: [e.g. Next.js 14, Tailwind CSS, Supabase, TypeScript]

Please:
1. Suggest the complete folder structure
2. List all dependencies I need to install
3. Create the main entry files with boilerplate
4. Identify the 3 most important components to build first
5. Point out any potential technical challenges early

Keep it production-ready and beginner-friendly.
```

### 🐛 The Bug Terminator
```
I have a bug and I can't figure it out. Here's everything:

**What I expected:** [describe expected behavior]
**What actually happens:** [describe actual behavior]
**Error message (if any):** [paste error]
**Relevant code:** [paste code]
**What I've already tried:** [list attempts]

Please:
1. Identify the root cause
2. Explain WHY it's happening in simple terms
3. Give me the fix
4. Tell me how to avoid this in the future
```

### 🎨 UI Component Generator
```
Create a [COMPONENT NAME] component for my [Next.js / React] app.

Requirements:
- Style: [e.g. modern, minimal, glassmorphism, dark mode]
- Must include: [list features]
- Props it should accept: [list props]
- Should be responsive: yes/no
- Animations: [subtle / none / bold]

Use Tailwind CSS. Make it production-ready and accessible.
```
